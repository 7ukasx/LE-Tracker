
# LE Tracker - High-End Personal Finance

Ein moderner, minimalistischer Finanz-Tracker mit Fokus auf Privatsphäre und Design.

## Features
- **Total Privacy**: Alle Daten werden lokal im Browser gespeichert (`LocalStorage`). Nichts verlässt dein Gerät.
- **Budgeting**: Setze dir monatliche Limits für verschiedene Kategorien.
- **Visualisierungen**: Schöne Graphen und Statistiken für deine Ausgaben.
- **Dark & Light Mode**: Nahtlose Umschaltung zwischen den Themes.
- **Automatisierung**: Smarte Fixkosten-Buchung und Gehaltseingänge.

## Rechtliches & Datenschutz
Da diese App keine Daten an Server sendet, ist sie extrem datenschutzfreundlich. Ein Impressum und eine Datenschutzerklärung sind direkt in der App unter den Profileinstellungen integriert.
